# Novox Blog Admin Studio

A premium, AI-driven administration dashboard for generating, verifying, and publishing SEO-optimized educational blogs for Novox Edtech.

## Key Features
*   **AI Drafting:** Integrates with Gemini 2.5 Flash to write search-intent driven copy.
*   **Real-time SEO Verification:** Client-side checks for keywords, headings, FAQ sections, and CTAs.
*   **Automated Git Pushes:** Direct commits and updates to destination websites using GitHub API.
*   **Imagen 4 Integration:** Automatic featured image generation.

## Local Setup
1. Run `npm install` to load dependencies.
2. Configure `.env` with your API keys and repo endpoints.
3. Start the dev server using `npm run dev`.
